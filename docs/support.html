Tappy Privacy Support

Support

Tappy support covers setup, Input Monitoring guidance, sound pack behavior, premium unlock issues, and general troubleshooting.

Common help topics

  • Enable Input Monitoring in System Settings → Privacy & Security.
  • Relaunch Tappy after changing Input Monitoring permission.
  • Re-select a pack in the carousel if you want to force a sound reload.
  • Use the built-in diagnostics panel if sound playback stops.

Contact channel

After the GitHub repository is published, this page should be updated with the public issue tracker or support email. For now, treat this page as the canonical support URL for App Store submission.

What to include in a report

  • macOS version
  • Tappy version
  • Which sound pack is selected
  • Whether Input Monitoring is enabled
  • What key behavior you expected and what happened instead
