Tappy Privacy Support

Support

Tappy support covers setup, Input Monitoring guidance, sound pack behavior, premium unlock issues, and general troubleshooting.

Common help topics

  • Enable Input Monitoring in System Settings → Privacy & Security so Tappy can play sounds while you type in other apps.
  • Relaunch Tappy after changing Input Monitoring permission if the app says restart is required.
  • Re-select a pack in the carousel if you want to force a sound reload.
  • Confirm system output volume and Tappy’s click volume are turned up if sound playback stops.

Contact

For support, report an issue at github.com/casstao1/tappy/issues. Include the details below so the issue can be reproduced.

What to include in a report

  • macOS version
  • Tappy version
  • Which sound pack is selected
  • Whether Input Monitoring is enabled
  • What key behavior you expected and what happened instead
