Support

Tappy support covers keyboard setup, feedback pack behavior, premium unlock issues, and general troubleshooting.

Common help topics

  • Enable Input Monitoring for Tappy in System Settings so Tappy can provide auditory feedback while you type in other apps.
  • If feedback stops after granting permission, quit and reopen Tappy so macOS applies the new permission.
  • Re-select a pack if you want to force a feedback cue reload.
  • Confirm system output volume and Tappy’s feedback volume are turned up if playback stops.

Contact

For support, report an issue at github.com/casstao1/tappy/issues. Include the details below so the issue can be reproduced.

What to include in a report

  • macOS version
  • Tappy version
  • Which feedback pack is selected
  • Whether Input Monitoring is enabled for Tappy
  • What key behavior you expected and what happened instead
