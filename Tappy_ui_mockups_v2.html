H · Full Icon Skin — every surface speaks keycap

The icon's three-layer language applied system-wide: sidebar = outer shell (darkest, most recessed), main area = cavity (mid-dark, inset shadow, ambient purple glow at the floor), every interactive tile = a frosted keycap surface. Two-column layout adds breathing room and makes each nav item a pressable key — very on-theme.

Reading the concepts: E is the lightest lift — just swap the pack tiles for icon-anatomy cards. F wraps the whole window in the icon shell (most immersive, lowest layout change). G promotes the detail card into a full keycap object while keeping the rest clean. H goes all-in: sidebar as shell, content as cavity, everything interactive is a keycap. All four share the same extracted icon tokens — shell #0d0b1a, cavity #181528, keycap face #c8c4e0, glow #7b5af0.
